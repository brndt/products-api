# products-api

## Environment setup
1. [Install docker](https://docs.docker.com/get-docker/)
2. Clone project with command `git clone https://github.com/brndt/products-api.git`
3. Move to the project directory: `cd products-api`

## Start project
```bash
make build
```

## Down project
```bash
make down
```

## Run test
```bash
make test-acceptance
```
